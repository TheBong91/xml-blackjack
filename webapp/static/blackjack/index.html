BlackJack

Blackjack, also known as twenty-one, is a comparing card game between a player and dealer, meaning players compete against the dealer but not against other players. It is played with one or more decks of 52 cards, and is the most widely played casino banking game in the world.

Technology

BaseX & XML

BaseX is a native and light-weight XML database management system and XQuery processor specialised in storing, querying, and visualizing large XML documents and collections.

XSLT

XSLT (Extensible Stylesheet Language Transformations) is a language for transforming XML documents into other XML documents,or other formats such as HTML, plain text, as PDF, PostScript or PNG

XQUERY

XQuery (XML Query) is a query and functional programming language that queries and transforms collections of structured and unstructured data, usually in the form of XML.

DOCBOOK

DocBook is a semantic markup language for technical documentation. It was originally intended for writing technical documents related to computer hardware and software.
